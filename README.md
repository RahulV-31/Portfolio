## Introduction
Welcome to my portfolio website! This project showcases my personal and professional information, including my resume, projects, and a contact form. It serves as an online resume and portfolio for potential employers and collaborators to learn more about me and my work.

## Features
- **Home Page**: Overview of who I am and what I do.
- **Resume Page**: Detailed resume with educational background, work experience, skills, and more.
- **Projects Page**: Showcase of my projects with descriptions and links to the project details or repositories.
- **Contact Page**: Form to get in touch with me directly via email.

## Technologies Used
- HTML5
- CSS3
- Font Awesome (for social media icons)
- GitHub Pages (for hosting)
